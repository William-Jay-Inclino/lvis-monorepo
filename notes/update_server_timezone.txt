Set the Timezone Using timedatectl
You can use the timedatectl command to set the timezone on your server. Hereâ€™s how:

List Available Timezones: First, you can list all available timezones to ensure that Asia/Manila is available:

timedatectl list-timezones | grep Manila

This should show Asia/Manila if it's available.

Set the Timezone: To change the timezone to Asia/Manila, run the following command:

sudo timedatectl set-timezone Asia/Manila
